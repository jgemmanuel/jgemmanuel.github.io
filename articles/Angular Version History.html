Angular Version History

2021-04-09 – J G Emmanuel

Angular 13

  • New: Modernization of the Angular Package Format (APF)
  • New: Test module teardown enabled by default
  • New: Persistent build cache by default
  • New: Correct parsing of query parameters by the default URL serializer
  • New: popstate event now emitted by simulateHashChange in SpyLocation under RouterTestingModule
  • New: FormControlStatus type
  • New: null / undefined values to routerLink disables navigation; DebugElement.properties['href'] now returns full URL
  • New: Router no longer replaces the browser URL when a new navigation cancels an ongoing navigation
  • New: SwUpdate#activateUpdate and SwUpdate#checkForUpdate now returns Promise<boolean>
  • New: Recursively check child routes on route detachment / attachment
  • New: import() now treats scripts as ES modules
  • Removed: View Engine
  • Removed: IE11 Support
  • Removed: Support for TypeScript versions older than 4.4.2
  • Removed: Support for NodeJS versions older than 12.20.0
  • Removed: WrappedValue class from @angular/core
  • Removed: Lazy routes using string for loadChildren and supporting classes to load NgModuleFactory at runtime

Angular 12

  • New: Ivy by default
  • New: Ivy-based IDE language service enabled by default
  • New: Angular CDK and Angular Material internally now use Sass module system
  • New: Critical CSS inlining enabled by default
  • New: ng build produce production bundle by default
  • New: <input type="number"> trigger min and max validation for formControl, formControlName, and ngModel directives
  • Removed: Support for TypeScript versions older than 4.2.3
  • Removed: Support for node-sass
  • Removed: Support for Webpack 4
